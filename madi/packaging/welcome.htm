This keyboard was developed to facilitate typing the Maʼdi language.

How to type in Maʼdi

ʼ – Modifier Letter Apostrophe
Used to for glotal stops and for implosives. Press ;
◌̣ – Dot Below
Used to indicate heavy vowel. Type a vowel then press [
◌́ – Acute
Used for high tone. Type a vowel then press ]
◌̃ – Tilde
Used for low tone. Type a vowel then press }
◌̂ – Circumflex
Used for falling tone. Type a vowel then press \

Mid tone is unmarked.

Fonts

The Charis SIL and Charis SIL Compact fonts are installed with the Maʼdi keyboard layout. These fonts are available under the Open Font License. Other suitable fonts include:

  • African Sans Serif
  • African Serif
  • Andika and Andika Compact
  • DejaVu Sans
  • Doulos SIL Compact and Doulos SIL
  • Gentium Basic and Gentium Book Basic
  • Gentium Plus and Gentium Plus Compact
  • Liberation Sans

Core fonts on WIndows 7 and Windows 8 should also be suitable.

The compact version of the SIL fonts have a tighter line spacing, so lines of text will appear closer together than when text is displayed using the original (non-compact) versions of the same font.

Literacy versions of Charis SIL, Charis SIL Compact, Doulos SIL, Doulos SIL Compact, Gentium Plus and Gentium Plus Compact are available.
