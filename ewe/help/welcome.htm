Typing additional characters

  1. Vowels with a grave: type the vowel then the ` key, e.g. ( a` yields à )
  2. Vowels with an acute: type the vowel then the \ key, e.g. ( a\ yields á )
  3. Vowels with a tilde: type the vowel then the ~ key, e.g. ( a~ yields ã )

Fonts

The Doulos SIL Compact and the Charis SIL Compact fonts are installed with the keyboard layout. Additional fonts include:

  • African Sans Serif
  • African Serif
  • Andika Basic
  • Charis SIL
  • DejaVu Sans
  • Doulos SIL
  • Gentium Basic and Gentium Book Basic

For information on installing fonts in Windows XP, please read the MyLanguage font installation guide.
